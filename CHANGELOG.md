# Changelog

All notable changes to the **VS Code Snow Blue Extension** are documented here.

This project follows [Keep a Changelog](https://keepachangelog.com/)
and uses [Semantic Versioning](https://semver.org/).

---

## [2.1.0] – 2026-04-19

### Added

- ⚡ **Snow Throne: Switch Theme** command
  - Switch between all themes directly from the Command Palette
  - No need to open the Color Theme picker manually

- 🌙 **Slate Amber Night (Premium Default)**
  - Built using a **60–30–10 color system**
  - Deep slate background for reduced eye strain
  - Structured blue-gray UI layers for clarity
  - Amber accent system for focus and highlights

- ❄️ **Snow Blue (Default)**
  - Refined snow-inspired dark base
  - Improved syntax hierarchy for web development
  - Reduced visual noise for long coding sessions

- 🎨 Unified **Design System**
  - Standardized 60–30–10 color rule across all themes
  - Consistent contrast behavior across dark and light modes

### Fixed

- Split merged `extension.ts` into `extension.ts` and `previewPanel.ts`
- Added missing `tsconfig.json`
- Added `commands` contribution to `package.json`
- Fixed `.vscode/launch.json` with `preLaunchTask` and `outFiles`
- Added `.vscode/tasks.json` for TypeScript watch/compile tasks

## [1.2.0] – 2025-01-30

### Added

- 🎄 **Christmas Day (Light Theme)**
  - Festive daytime coding theme
  - High readability in bright environments
  - Traditional Christmas color palette

- 🌙 **Christmas Night (Dark Theme)**
  - Cozy festive dark theme for nighttime coding
  - Warm accents optimized for low-light environments

- 🎨 Christmas Color System
  - Cinnabar (#EA4630)
  - Firebrick (#BB2528)
  - Christmas Green (#146B3A)
  - Orange-Yellow (#F8B229)
  - Chinese Yellow (#FFAE00)
  - Floral Lavender (#AF7AFF)
  - Sunset Orange (#FF6058)

- 📦 Multi-theme extension structure
  - Snow Blue + Christmas variants bundled together

### Improved

- Enhanced theme separation (base vs seasonal themes)
- Better readability for HTML, CSS, and JavaScript
- Improved contrast tuning across all themes
