# Changelog

All notable changes to the **Vscode Snow-Blue** theme are documented here.

This project follows [Keep a Changelog](https://keepachangelog.com/)
and uses [Semantic Versioning](https://semver.org/).

---

## [1.0.4] – 2025-01-28

### Added

- Marketplace-ready extension icon
- GitHub Actions workflow for automated publishing
- License file (MIT)
- Improved README with installation and usage instructions

### Improved

- HTML readability:
  - Clear separation between tag names, attributes, and values
  - `class="..."` values are easier to scan without overpowering
- CSS readability:
  - Strong selector → property → value hierarchy
- JavaScript readability:
  - Better distinction between built-in APIs and user-defined functions
- Reduced glare and balanced contrast for long coding sessions

---

## [1.0.3] – 2025-01-27

### Added

- Initial Snow-Blue dark theme
- Snow-Throne color philosophy (snow → carbon ramp)
- Optimized defaults for web development (HTML / CSS / JS)
- Markdown-friendly styling for notes and learning

---

## [Unreleased]

- Ongoing tuning for contrast and accessibility
